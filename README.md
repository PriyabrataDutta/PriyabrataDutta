## 🧠 Engineering Philosophy

```
🎯 Start with the problem, not the tech.
   → Chose MySQL full-text over Elasticsearch for 5K products — right tool at that scale.

🔬 Separate deterministic from probabilistic.
   → In AI systems, data retrieval is testable. LLM analysis is not. Keep them apart.

🚢 Ship, then optimize.
   → First versions use the simplest architecture that works. Complexity is earned by real traffic.

📖 Code is read more than it's written.
   → If a PR needs a comment to explain logic, the logic needs rewriting.
```
